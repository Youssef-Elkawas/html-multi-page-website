# HTML Multi-Page Website

This is my first structured multi-page website built using pure HTML.

## 📖 About This Project

After finishing HTML fundamentals, I wanted to build something practical instead of just watching tutorials.

So I built a complete 5-page static website from scratch.

## 🛠 What I Practiced

- Semantic HTML structure
- Navigation between multiple pages
- Forms and input fields
- Lists and tables
- Content organization using sections and articles
- Proper document structure (header, main, footer)

## 📂 Pages Included

- Home
- About
- Blog
- Projects
- Contact

## 🎯 Why I Built This

The goal was not design.
The goal was structure.

I focused on writing clean, meaningful HTML before moving to CSS.

## 🚀 Next Step

Now I’m moving to CSS to style everything and improve the visual presentation.
